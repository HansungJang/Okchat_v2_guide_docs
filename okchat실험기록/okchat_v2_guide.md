# Okchat v2 챗봇 가이드

본 가이드는 처음 Okchat v2에 가입한 사용자가 손쉽게 챗봇을 생성하고, 서비스 구성, 학습, 그리고 다양한 외부 연동을 통해 챗봇을 효율적으로 활용할 수 있도록 돕는 지침서입니다.

## 1. 챗봇 생성하기

### 1.1 개요

- 신규 가입 사용자를 위한 챗봇 생성 프로세스 안내

### 1.2 기존 챗봇 작업 이어가기

- 이미 작업 중인 챗봇이 있을 경우, 해당 챗봇 관리 화면으로 이동하는 방법

### 1.3 새로운 챗봇 생성

- 기본 설정: 챗봇 이름, 언어 설정, 기본 테마
- 초기 지식 베이스 설정(선택 사항)

## 2. 서비스 구성

챗봇 생성 후 제공되는 다양한 서비스 페이지와 기능 설정에 대해 살펴봅니다.

### 2.1 개요

- **대시보드**:
  - 월별 사용자 수, 잔여 크레딧 확인
  - 챗봇 이용 현황 모니터링

### 2.2 대화

- **데모**:
  - 작업한 챗봇의 데모 버전 테스트하기
  - 주의사항: Refresh 미적용 시 이전 대화기록이 이후 응답에 영향을 줄 수 있음
- **히스토리**:
  - 해당 챗봇을 통해 이루어진 모든 사용자 대화 기록 열람
- **리드**:
  - 사용자 정보 수집 동의 고객 정보 확인
  - 사용자 이력, 활용 현황 파악

### 2.3 지식

- **지식 베이스**:
  - 챗봇에 자료를 추가하여 학습시키는 기능
  - 자료 유형:
    - URL 크롤링 (횟수 제한, 주의사항)
    - Text 데이터
    - Markdown, PDF 파일
- **학습 데이터**:
  - 학습한 자료 목록 확인 및 삭제
  - (수정 기능은 추후 제공 예정)

### 2.4 통합

- **도구 (API 연결)**:
  - 외부 API 연결 방법 및 설정
- **소셜 미디어 연동**:
  - 네이버, 카카오톡, 인스타그램 등 다양한 플랫폼 연동 가이드
- **채팅 위젯**:
  - 챗봇 로고, 대표 색상, 디자인 요소 변경
  - 웹사이트나 서비스 내부에 위젯 삽입 가이드

### 2.5 설정

- **일반 설정**:
  - 챗봇 응답 형식/스타일, 대표 색상, 기본 Prompt 설정
- **고급 설정**:
  - 사용자 히스토리 관리 정책
  - 지식 데이터 백업/내보내기 옵션

## 3. 챗봇 학습하기

- 지식 베이스 활용법
- 4.1 도구(API), 4.3 채팅 위젯, 5.1 일반 설정을 활용한 챗봇 학습 방법
- 학습 시 주의사항 및 베스트 프랙티스

## 4. API 연동하기

- 준비 사항: API 키, 인증 정보
- 요청 형식: JSON 포맷, 엔드포인트 URL
- 요청 검증 방법: 올바른 요청인지 확인하는 절차
- 무료 API 리소스 및 참고 자료

## 5. 소셜 미디어 연결하기

- 준비 사항: 각 플랫폼별 개발자 계정, 권한 설정
- 연결 작업 가이드 영상 및 단계별 안내
- 주의사항: 토큰 갱신, 권한 관리, 사용자 정보 수집 관련 정책
